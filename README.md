* Helpdesk Home Lab
  This lab demostrates the setup of a virtual machine and basic windows help desk troubleshooting tasks.

* Tasks Performed
- Set up and configured Windows virtual machine  
- Created and managed a local user account  
- Configured shared folder 
- Monitored system performance using Task Manager  
- Verified network and connectivity settings  

*Skills Demonstrated
- Virtual machine setup (VirtualBox)  
- Windows user account management  
- File sharing and permissions  
- Basic troubleshooting and system monitoring  
- Networking basics


All screenshots from the lab are included in this repository.
